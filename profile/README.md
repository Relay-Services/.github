# Relay

Relay is a new web tools project built for fast access, simple controls, and services that do what they need to do without getting in the way.

The project is still early, but the plan is to build Relay as a connected platform of web tools. Some parts will be available without an account, while features that need saved settings, longer sessions, uploads, or private access will use Relay ID.

## What we are building

### Relay Web

Relay Web is the main access tool for Relay. It is being built around speed, clean sessions, and a simple interface that is easy to use from the start.

### Relay Drop

Relay Drop is for quick file uploads and share links. The goal is to make sending files simple without turning it into a complicated storage platform.

### Relay Live

Relay Live is planned for shared sessions and live browsing tools. It will be built after the core Relay services are more stable.

### Relay Chat

Relay Chat is planned as a basic communication tool for rooms and direct conversations across Relay.

## Platform systems

### Relay ID

Relay ID is the account system used across Relay. It will handle sign in, sessions, saved preferences, and access to features that need an account.

### Relay Panel

Relay Panel is the management side of Relay. It is used for platform controls, service settings, user tools, and internal administration.

## What matters to us

Relay is being built with a few main goals in mind:

* fast pages
* simple layouts
* secure accounts
* reliable services
* clear controls
* tools that work well together

## Status

Relay is in early development.

The current focus is the homepage, Relay Web, Relay ID, and the base structure that the rest of the platform will use.

## About

Relay is built for people who want web tools that are quick, understandable, and dependable. The project is starting small, but it is being designed so each service can grow without making the platform harder to use.
